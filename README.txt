# Descrição do Projeto

O projeto consiste no desenvolvimento de um sistema simples de ERP (Enterprise Resource Planning) voltado para o gerenciamento de pedidos de uma fábrica de cortinas.

O sistema foi desenvolvido utilizando JavaScript puro, com foco na aplicação prática dos conceitos fundamentais de lógica de programação estudados durante a disciplina.

O objetivo principal é simular o funcionamento básico de um ambiente empresarial, permitindo o cadastro e gerenciamento de pedidos de produção de cortinas tradicionais e motorizadas.

---

# Objetivos do Sistema

* Simular o controle de pedidos;
* Aplicar estruturas condicionais;
* Aplicar estruturas de repetição;
* Utilizar arrays e objetos;
* Trabalhar com funções;
* Organizar o código em módulos lógicos;
* Simular um pequeno sistema ERP de produção.
